## ✉️ Commit Convention
`타입: 부연 설명 및 이유 #이슈번호` `ex. Feat: Home 화면 UI 구현 #1`

```
🎉 Feat: 새로운 기능 추가
🔧 Fix: 버그 수정
📦️ Build: 빌드 관련 파일 수정
✅ Chore: 그 외 자잘한 수정
⚗️ Ci: CI관련 설정 수정
📝 Docs: Wiki, README 문서 (문서 추가 수정, 삭제)
🎨 Style: 스타일 (코드 형식, 세미콜론 추가, 비즈니스 로직 변경X)
♻️ Refactor: 리팩토링 (네이밍 변경, 포함)
🩹 Test: 테스트 코드 (추가, 수정, 삭제)
💥 Remove: 코드 삭제
```
